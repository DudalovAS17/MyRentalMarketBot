## Модель-User 99%

**удобное «отображаемое имя» (не хранится в БД)**
```
@property
def display_name(self) -> str:
    if self.full_name:
        return self.full_name
    if self.first_name or self.last_name:
        return f"{self.first_name or ''} {self.last_name or ''}".strip()
    return self.username or self.telegram_id
```

Это нужно будет вынести в 
- в DTO (UserOut.display_name вычисляемое)
или 
- в helpers/formatters

в виде:
```
from typing import Optional

def format_user_display_name(
    *,
    telegram_id: str,
    username: Optional[str] = None,
    first_name: Optional[str] = None,
    last_name: Optional[str] = None,
    full_name: Optional[str] = None,
) -> str:
    if full_name:
        return full_name
    if first_name or last_name:
        return f"{first_name or ''} {last_name or ''}".strip()
    return username or telegram_id

```
